# 🎮 Game Development Project

> A game development project built to explore modern game programming concepts, interactive gameplay mechanics, and software engineering principles.


---

## 📖 Overview

This repository contains a game development project that demonstrates core game programming concepts such as:

- 🎯 Interactive gameplay
- 🎨 Game graphics and animations
- 🎮 Player controls
- 🧠 Game logic
- 🔊 Audio integration
- 💾 Score/Progress management

The project was developed as a learning and portfolio project to strengthen practical game development skills.

---

## ✨ Features

- Smooth player movement
- Interactive gameplay mechanics
- Collision detection
- Score tracking
- Sound effects & background music
- Responsive UI
- Multiple game states (Menu, Play, Pause, Game Over)

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/YaS16s/Game-devlopment.git
```

### Open the project

If using **Unity**

```
Open Unity Hub
→ Add Project
→ Select this repository
→ Open
```

If using another engine, open the project using the appropriate IDE or game engine.
